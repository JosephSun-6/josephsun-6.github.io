    这一节谈谈秩和检验和效应量。并用R语言去进行Mann-Whitney U 检验和效应量检验。

    秩和检验

    1、配对资料的秩和检验：Wilcoxon符号秩检验

    2、两样本比较的秩和检验：Wilcoxon秩和检验(Mann-Whitney U test) —— 计量资料、等级资料

    3、完全随机设计多样本比较的秩和检验：Kruskal-Wallis H检验 —— 计量资料、等级资料

    4、随机区组设计资料的秩和检验：Friedman M检验

    公众号仅更新了Mann-Whitney U test，此类最为常用，其他的可以参考《非参数检验》这本书。

    两样本比较的秩和检验：Mann-Whitney U test

    秩和检验方法最早是由维尔克松(Wilcoxon)提出，叫维尔克松两样本检验法。后来曼—惠特尼将其应用到两样本容量不等(n1不等于n2)的情况，因而又称为曼—惠特尼U检验(Mann-Whitney U test)。这种方法不要求满足任何分布，基于样本构造检验统计量，用来检验两个独立样本是否来自同一个总体或者用于比较两个独立样本的差异。通俗来说是通过推断总体的分布相同与否来判断样本是否存在差异。

    其原理是将数值转换成次序，混合排序，对异常值不敏感，只关心相对大小排序，不关心绝对。

    优点是不受总体分布限制，适用面广；适用于等级资料及两端无确定值的资料；易于理解，易于计算。缺点是符合参数检验的资料，倘若采用用秩和检验，则不能充分利用信息，检验功效低。

    步骤：

    第一步，两个样本混合排序

    第二步，计算每个样本秩的和，排列后的次序相加

    第三步，计算统计量
    $$
    U_1=R_1-\frac{n_1(n_1+1)}{2}
    $$

    $$
    U_2=R_2-\frac{n_2(n_2+1)}{2}
    $$

    小样本时利用Wilcoxon rank sum statistic分布精确计算p-value

    大样本：如果$n_1>10$或$n_2>20$则可以采用正态近似法：
    $$
    W = \frac { | T - n _ { 1 } ( N + 1 ) / 2 | - 0.5 } { \sqrt { n _ { 1 } n _ { 2 } ( N + 1 ) / 12 } }
    $$
    相同秩次较多时的校正：
    $$
    u = \frac { | T - n _ { 1 } ( N + 1 ) / 2 | - 0.5 } { \sqrt { \frac { n _ { 1 } n _ { 2 } ( N + 1 ) } { 12 } ( 1 - \frac { \sum ( t _ { j } ^ { 3 } - t _ { j } ) } { ( N ^ { 3 } - N ) } ) } }
    $$

    另外，我还想对提一下wilcox.test()函数。

    它可以进行Wilcoxon符号秩检验，即wilcox.test(数据集，中位数，alternative = “less”或者”great”或者”two.sided”，exact = TRUE此项目的为是否精确计算P值，paired=TALSE此项是逻辑变量说明是否为成对数据，conf.int是逻辑变量用于是否给出相应的置信区间)。

    也可以进行非成对样本的秩次和检验 — Mann-Whitney U 检验，wilcox.test(数据集1，数据集2，…）

    解释结果时，先判断样本是否来自同一个总体样本，样本数目是否成对，非成对样本连续修正后结果的$p-value$与$\alpha$关系。套路为P值=…<[或>]0.05，拒绝原假设[或无法拒绝原假设]，认为两者效果(作用因素、人群生理特征等等)差异显著[或无显著差异]。

    Effect Size (效应量)

    Effect size is a statistical concept that measures the strength of the relationship between two variables on a numeric scale

    是衡量处理效应大小的指标，与显著性检验不同，这些指标是不受样本容量影响的。它表示不同处理下的总体均值之间差异的大小，可以在不同研究之间进行比较。一般用于针对某一研究领域内的元分析中。

    效应量（effect size）是对现象量级的定量度量。其包括两个变量之间的相关性，回归中的回归系数，平均差，甚至是发生某事的风险：如有多少人在心脏病发作后幸存下来。

    效应量是对统计检验的补充，可以评价统计的强度。效果量的结果有助于解释研究的实质意义，而不是统计意义的显著性。各个论文中不同时间地区的结论可以相互比较的方法，计算不同效应量，然后得出对比结论。

    换句话说，效应量就是说明p值显著不显著，采用”度”描述，显著程度。

    Meta - analysis (元分析)

    统计学上来说，元分析（meta-analysis，或译作后设分析、整合分析、综合分析、统合分析、荟萃分析）是指将多个研究结果整合在一起的统计方法。就用途而言，它是文献回顾的新方法。

    文献回顾的传统方法是叙事式的，由作者自行挑选觉得重要的前人研究，当各研究结论冲突时，由作者自行判断哪一种结论较具价值。反之，后设分析采用系统化的资料收集和数值分析，让文献回顾更具客观性与证据力。这些组合后的结果会有助于以更加全面性的证据观察来支持或拒绝某一主题研究的假设，发现该主题迄今研究的不足处，发现各种可能影响效应量的中介因素。

    P值只能提供有限的信息，我们要知道的是误差有多大，相关性有多强，而效果量可以告诉我们这些。例如，当我们说“吸烟和一个人生活满意度显著相关。（p <0 .001）”，但我们不知道它们的关联性有多强，如果我们发现相关系数是0.1，那么实际上它们相关性是挺弱的；又如果相关系数是0.7，那么实际上它们彼此间的关系是挺有趣的。

    Lakens (2013)中提到了关于各种指标的用途。

    常用的效应量包括r, Cohen’s d和odds ratio。对于T检验的效应量一般常用 Cohen’s d 效应量指标。
    $$
    Cohen’s d = (M_2 - M_1)/SD_{pooled}
    $$

    $$
    SDpooled 为联合方差 ： SD_{pooled} =\sqrt {\frac{SD^2_1+SD^2_2}{2}}
    $$

    $$
    或 \ \ \ d =\frac{|\overline x_1-\overline x_2|} {\sqrt{\frac{(n_1-1)s^2_1+(n2-1)s^2_2}{n_1+n_2-2} }}
    $$
