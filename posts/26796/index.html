arcgis api for js 部署

发表于 2020-05-25 更新于 2020-06-18 分类于 编程 阅读次数：

部署arcgis api for js

IIS设置

打开控制面板-程序-启动或关闭windows功能-勾选 internet information services

然后在windows图标那里点搜索IIS

打开MIME类型把没有的添加进去

arcgis api 下载途径

esri官网下载，下载需要注册 arcgis 用户才行，版本自己选择，官网下载最新版本api。

IIS部署配置方案(以3.27为例)

  1. 把下载的arcgis api for js 3.27 离线包解压拷贝到 C:\inetpub\wwwroot 目录下。
  2. C:\Inetpub\wwwroot\arcgis_js_api\library\3.27\init.js在文本编辑器中打开并搜索文本"https://[HOSTNAME_AND_PATH_TO_JSAPI]dojo"，然后将其替换为"http://hocalhost/arcgis_js_api/library/3.27/dojo"
  3. C:\Inetpub\wwwroot\arcgis_js_api\library\3.27\dojo\dojo.js在文本编辑器中打开并搜索文本"https://[HOSTNAME_AND_PATH_TO_JSAPI]dojo"，然后将其替换为"http://hocalhost/arcgis_js_api/library/3.27/dojo"
