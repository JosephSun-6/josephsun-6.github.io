地图量算

发表于 2020-05-25 更新于 2020-06-18 分类于 编程 阅读次数：

地理毕竟是理科，大部分时候涉及数学，尤其长、宽、面积、周长、形心等等。

有三种方法用来进行地图量算。

1、Arcgis for server 中调用 Geometry 方法

arcgis for server中自带有地图量算工具

https://localhost:6443/arcgis/rest/services/Utilities/Geometry/GeometryServer （端口设置可能不一样）

利用函数Areas and Lengths，其中 Areas 用于量算面积，Lengths 用于距离。

2、geometryEngine 接口

arcgis api 提供geometryEngine

接口函数。

geodesicArea(geometry, unit) 用于计算面积

geodesicLength(geometry, unit) 用于计算距离

3、创建自定义类

这样的好处是不依赖ArcGIS Server 几何服务 Geometry 以及arcgis api 接口类 geometryEngine，灵活应用在WebGIS 项目中。
